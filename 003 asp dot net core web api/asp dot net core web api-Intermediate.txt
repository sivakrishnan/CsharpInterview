###1) What is model binding?

	[FromHeader] → bind from headers

###4) What is the difference between IActionResult and ActionResult<T>?

	IActionResult is a flexible interface that lets you return any type of HTTP response, 
	while ActionResult<T> combines type safety with flexibility. With ActionResult<T> you can return either a strongly typed object (which automatically becomes a 200 OK response) 
	or other HTTP responses like NotFound() or BadRequest(). It also improves Swagger/OpenAPI documentation by clearly showing the response type.


	[HttpGet("{id}")]
	public IActionResult GetProduct(int id)
	{
		var product = _repository.GetById(id);

		if (product == null)
			return NotFound();

		return Ok(product);
	}
	
	[HttpGet("{id}")]
	public ActionResult<Product> GetProduct(int id)
	{
		var product = _repository.GetById(id);

		if (product == null)
			return NotFound();

		return product; // automatically wrapped in Ok(product)
	}

###12) What is content negotiation?

	Content negotiation is the process where the server and the client agree on the best format for the response data.
	in short server and client agree on data format dynamically.
	
	If a client sends:
	Accept: application/json
	the server responds with JSON.
	example: 
	
		{
		  "id": 1,
		  "name": "Phone"
		}
	
	If a client sends:
	Accept: application/xml
	the server responds with XML
	example: 
		<Product>
		  <Id>1</Id>
		  <Name>Phone</Name>
		</Product>
	
	in program.cs
	
	builder.Services.AddControllers().AddXmlSerializerFormatters();

###15) Explain versioning of Web APIs.

	1) URI versioning (most common)
		/api/v1/products
		/api/v2/products
		
	2) Query string versioning		
		/api/products?api-version=1.0
		
	3) Header versioning
		api-version: 1.0
	
	4) Media type versioning 
		Accept: application/vnd.myapp.v1+json
		
	Microsoft provides a great NuGet package:Microsoft.AspNetCore.Mvc.Versioning
	
	in program.cs file
	
		builder.Services.AddApiVersioning(options =>
		{
			options.DefaultApiVersion = new ApiVersion(1, 0);
			options.AssumeDefaultVersionWhenUnspecified = true;
			options.ReportApiVersions = true;
		});
